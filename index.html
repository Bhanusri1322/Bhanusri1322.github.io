Project Title 2

Hybrid MIMO Detection Algorithm

This project is about developing a hybrid algorithm for detecting signals in Large MIMO systems, which are used in modern wireless communications. The goal is to make signal detection faster and more efficient while keeping the complexity low. I combined an optimization technique called IAMMPSO (Improved Adaptive Multi-Population Particle Swarm Optimization) with traditional MIMO detection methods. This approach improves both speed and accuracy, making it suitable for real-time applications.
